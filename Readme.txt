1. Download the Hadoop virtual machine for VMWare from the following website:
http://ydn.zenfs.com/site/hadoop/hadoop-vm-appliance-0-18-0_v1.zip

2. Install the Virtual Machine

3. Install the Hadoop plugin for Eclipse

4. Setup MapReduce location in Eclipse to point to HDFS running on your virtual machine

5. Run <JoinMapReduce.java> as MapReduce job
